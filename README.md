# 鍛冶屋シミュレーター (Rustic Forge)

## 概要
ブラウザ上で動作する本格的な「鍛冶屋シミュレーター」です。
プレイヤーは辺境の街の鍛冶屋となり、素材を集め、武具をクラフトし、冒険者たちに販売します。100日後に訪れる「破壊神」の脅威から街を守り抜き、富と名声を得ることが目的です。

## 特徴
- **クラフトシステム**: 素材を組み合わせて武器・防具・盾を作成。品質（粗悪〜エピック）やエンチャント（特殊効果）がランダムで付与されます。
- **冒険者との取引**: 冒険者には適正重量や得意な装備カテゴリがあり、適切な装備を良心的な価格で販売することで好感度と名声が上昇します。
- **ダンジョン探索と街の防衛**: 装備を整えた冒険者はダンジョンへ向かい、素材と資金を持ち帰ります。一定日数ごとに襲来するボスに対しては、街の総戦力で迎え撃ちます。
- **美麗なビジュアル**: AIによって生成されたハイクオリティなファンタジー背景と、モダンなグラスモーフィズムUIを採用しています。
